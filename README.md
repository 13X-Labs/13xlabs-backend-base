# 13XLabs


## Install

### Create Virtual Environment
```bash
$ python3 -m venv venv
```

### Active Virtual Environment
```bash
$ venv/Scripts/activate (Windows)
$ source venv/bin/activate (Linux)
```

### Requirements
```bash
$ pip install -r requirements.txt
```

### Run
```bash
$ python manage.py runserver
```

## Development

### Freeze Requirements
```bash
$ pip freeze > requirements.txt
```
